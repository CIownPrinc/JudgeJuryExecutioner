# GAVEL v0.1 Vertical Slice

A polished browser-playable vertical slice of **GAVEL: A Game of Judge, Jury, and Executioner**.

## Run locally

```bash
npm run dev
```

Then open http://127.0.0.1:5173.

## What is included

- One complete 3–5 minute case loop: intake, investigation, deliberation, verdict, sentencing, results, and play again.
- Randomized defendants, charges, evidence order, hidden truth, witness reliability, and case summaries.
- Push-your-luck Court Time during investigation.
- Evidence weighting and fuzzy gut-signal feedback during deliberation.
- Risk-weighted sentencing outcomes with Authority and Integrity feedback.
- No backend, database, accounts, external APIs, progression, or save data.
